!Dataset is on SSD!


Each directory sequence_x consists of multiple sequences depicting human body in various poses. 
Poses are stored sequentally in order. Each pose is captured from multiple views and is stored in camera_y directory
apropriate skeletons are found in skeletons directory. 
Point clouds are stored in format H x W x 3. 
Poses in format 20 x 3
(2 * (heels, ankle, knee, pelvis), 2 * (hand, elbow, shoulder), head, neck, 4 * back joints)
